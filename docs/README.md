## Adding tasks
There are 3 types of tasks you may add:
1. ToDo
2. Deadline
3. Event

## ToDo
Usage: `todo [taskName]`

## Deadline
Usage: `deadline [taskName] /by [date]`

## Event
Usage: `event [taskName] /from [date] /to [date]`
