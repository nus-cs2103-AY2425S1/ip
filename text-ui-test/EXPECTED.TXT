 What can I do for you? Here are the available commands:

 1. todo [task]: Adds a new todo task
   Example: todo buy groceries

 2. deadline [task] /by [date]: Adds a task with a deadline
   Example: deadline submit report /by 2024-09-15

 3. event [task] /from [start time] /to [end time]: Adds an event with start and end times
   Example: event project meeting /from 2024-09-10 14:00 /to 2024-09-10 16:00

 4. list: Lists all tasks

 5. mark [task number]: Marks a task as done
   Example: mark 2

 6. unmark [task number]: Unmarks a completed task
   Example: unmark 2

 7. delete [task number]: Deletes a task
   Example: delete 3

 8. find [keyword]: Finds tasks that match the given keyword
   Example: find meeting

 9. bye: Exits the application
