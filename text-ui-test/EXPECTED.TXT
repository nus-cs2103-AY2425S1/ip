Here is a list of commands to get you started:
bye - Exits the Ginger chatbot.
list - Lists the current tasks.
mark - Marks a task as complete. Usage: mark <task number>
unmark - Marks a task as incomplete. Usage: unmark <task number>
todo - Adds a To Do task. Usage: todo <title>
deadline - Adds a Deadline task which includes a deadline. Usage: deadline <title> /by <time>
event - Adds an Event task which has start and end time. Usage: event <title> /from <time> /to <time>
help - Displays this set of information again.
delete - Deletes a task from the list. Usage: delete <task number>

Do note that Ginger supports date and time formatting. Please follow the convention dd/mm/yyyy HHmm
when entering your time! Examples: 01/08/2024 1830, 1/8/2024 1830 is also accepted.
